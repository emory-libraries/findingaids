About/welcome

This database searches detailed descriptions of manuscript collections, which are called finding aids. Finding aids generally include:

  • A description of the collection's creator
  • A description of the contents
  • A list of boxes and folders in which materials are stored

Please note: this site includes finding aids for unprocessed collections.
